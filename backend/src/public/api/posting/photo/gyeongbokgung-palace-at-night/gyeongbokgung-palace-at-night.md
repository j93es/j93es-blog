## Intro

경복궁 야간 개장에 다녀왔습니다. 야간에서는 사진 찍기가 정말 어렵더군요. 잡광/노이즈를 잡는데 정말 애먹었습니다. 더하여 조리개 최대개방에서도 사진이 어두워서, 셔터스피드를 낮추면 사진이 흔들리고... 하지만 여러 조건 하에서 사진을 찍고, 좋은 결과물을 만드려내려는 노력, 그 과정 자체는 정말 재밌는 것 같습니다.

- EOS R8 + RF 35mm f1.8
